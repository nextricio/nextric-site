# Nextric.io - AI-Powered Passive Income Insights

A modern, fast Jekyll site for publishing SEO-optimized content.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000

## Auto-Deployment

Articles placed in `_posts/` are automatically published when pushed to main branch.

## Structure

- `_posts/` - Blog articles (YYYY-MM-DD-title.md format)
- `_drafts/` - Work in progress
- `_layouts/` - Page templates
- `_includes/` - Reusable components
- `assets/` - Images, CSS, JS
- `keywords/` - Keyword research data
